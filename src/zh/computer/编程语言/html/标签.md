## 标签

      
| 标签 | 描述 | 例子 |
| :---: | :---: | :--- |
| `<h>` | 标题 | `<h1><h1/>`|
| `<p>` | 段落 | `<p><p/>`|
| `<br>`| 换行 | `<br>`|
| `<hr>`| 水平线 | `<hr>`|
| `<! >`| 注释 | `<! 这是一个注释>`|
| `<a>` | 链接 | `<a href="https://www.runoob.com">这是一个链接</a>`|
| `<img>`| 图像 | `<img src="/images/logo.png" width="258" height="39" />`|

HTML 文本格式化标签

| 标签 | 描述 |
| :--- | :--- |
| `<b>` | 定义粗体文本 |
| `<em>` | 定义着重文字 ( 斜体字 ) |
| `<i>` | 字体图标 |
| `<u>` | 下划线 |
| `<small>` | 定义小号字 |
| `<strong>` | 定义加重语气 |
| `<sub>` |	定义下标字 |
| `<sup>` |	定义上标字 |
| `<ins>` |	定义插入字 |
| `<del>` |	定义删除字 |
